## Chappy

Länk till Publicerad sida:
https://fullstack-setup-1.onrender.com/

Välkommen till projektet Chappy!
Här har jag jobbat med både backend och frontend.

I Chappy kan man:
- chatta i öppna och låsta kanaler
- skicka DM (inloggad)
- registrera sig
- logga in
- skapa egna kanaler (inloggad)
- ta bort egna kanaler (inloggad)
- ta bort sitt konto (inloggad)

Tekniker jag har använt:
## Frontend
- React + Vite + TypeScript
- React Router
- Zustand
- Fetch API
- CSS
- Backend
- Node + Express
- TypeScript
- JWT
- Bcrypt
- DynamoDB
- Zod
- Insomnia

## Installation
git clone https://github.com/Juliahagstedtt/chappy.git
cd chappy
npm install

## Starta Backend:
npm run build-server
npm run start-server

## Starta Frontend:
npm run dev
